## Component Structure Diagram

```
Header.tsx
├─ Logo + Brand
├─ Navigation Links (hidden on mobile)
├─ Dark Mode Toggle
└─ Mobile Menu Button

Layout.tsx
├─ Header
├─ Sidebar (hidden < lg)
├─ Main Content Area
│  └─ {children}
└─ Footer

Sidebar.tsx
├─ Table of Contents
│  └─ NavItem Links
└─ Progress Indicator

SectionCard.tsx
├─ Title Section (Header)
│  ├─ Vietnamese Title (H2)
│  └─ English Title (italic)
├─ Content Section
│  ├─ Assignment (blue)
│  ├─ Reading (amber)
│  └─ References (list)
└─ Footer
   ├─ Section Number
   └─ Share Button

Breadcrumb.tsx
├─ Item 1
├─ Separator
├─ Item 2
└─ Separator + Current Item (bold)

Footer.tsx
├─ About Section
├─ Resources Links
├─ Support Links
├─ Legal Links
└─ Copyright
```

## Responsive Breakpoints

### Mobile (< 640px)
- Single column
- Full-width content
- Hamburger menu
- Sidebar hidden
- Stacked cards

### Tablet (640px - 1024px)
- Two columns where applicable
- Sidebar hidden
- Optimized spacing
- Touch-friendly buttons

### Desktop (1024px+)
- Three-column layout (sidebar + main + optional)
- All features visible
- Maximum readability
- Sidebar always visible

---

## Interactive States

### Hover Effects
```
Link/Button:    text-color change + slight opacity
                transition: 200ms ease
                
Card:           shadow increase
                transition: 200ms ease
                
Navigation:     background color change
                text color change
```

### Focus States
```
Button:         outline ring-2 ring-offset-2
Link:           underline + outline
Form:           ring around input
```

### Active States
```
Navigation:     bold text + background color
Breadcrumb:     last item bold
Progress:       filled portion highlighted
```

## Typography Hierarchy

```
H1 (4-5xl, bold)
  Primary page title

H2 (2xl, bold)
  Section heading

H3 (lg, semibold)
  Subsection

H4 (base, semibold)
  Card title

Body (base, regular)
  Main content

Small (sm, regular)
  Secondary info

Tiny (xs, regular)
  Meta data
```
